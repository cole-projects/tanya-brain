# Client Sessions

> Index for all 1:1 coaching session transcripts.
> Each client has their own folder inside `02-Client-Sessions/` with sessions named by date.
> Use this file to navigate by client, identify recurring themes, and connect session work back to frameworks.

---

## How to Use This Index

- **For a specific client:** Go to their folder — read their `profile.md` first, then the relevant session
- **For a recurring theme:** Use the Theme → Framework map at the bottom
- **Before responding to a client:** Read their `profile.md` + check [[Client-Response-Protocol]] for triage guidance
- **After a new session:** Update the client's `profile.md` with anything new that emerged
