## Adding a New Client

1. Create a session folder: `02-Client-Sessions/[FirstName]/`
2. Add session file named by date: `YYYY-MM-DD.md`
3. Once they have 2+ sessions, create a profile using [[Templates/Client-Profile-Template]]
4. Save profile to `02-Client-Sessions/Client Profiles/[FirstName].md`
5. Add them to this hub and update [[02-Client-Sessions]]
